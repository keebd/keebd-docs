# Kit Content

The Discipad is a 17-key wired numpad with USB Type-C. It's a fully through-hole
kit, meaning you solder every component yourself rather than just mounting a
controller board.

Here's a list of parts needed for the build:

* 1 x [Discipad Numpad Keyboard Kit](https://keebd.com/products/discipad-numpad-keyboard-kit)

  Kit Includes:
  * 1 x Discipad PCB
  * 1 x ATmega328p Microcontroller
  * 1 x 28-pin IC Socket
  * 17 x 1N4148 Diodes
  * 1 x USB Type-C Port
  * 2 x 3.6V Zener Diodes
  * 1 x Resettable Fuse (5.1mm)
  * 1 x 16MHz Crystal
  * 2 x 0.1uF Capacitors
  * 2 x 22pF Capacitors
  * 1 x 4.7uF Capacitor
  * 1 x 10K Resistor
  * 2 x 5.1K Resistors
  * 1 x 1.5K Resistor
  * 2 x 75R Resistors
  * 1 x 6-pin Header
  * 2 x 6mm Pushbuttons (Boot & Reset)
  * 4 x M2 4mm Screws
  * 4 x M2 5mm Standoffs

Not included in the kit but required to complete:
* 17 x [PCB or Plate Mount MX-style Switches](https://keebd.com/collections/switches)
* 17 x [Keycaps for MX Switches](https://keebd.com/collections/caps)
* 3 x 2u Screw-in Stabilisers
* 1 x USB Type-C Cable

Tools required:
* Soldering iron and solder (Kester 63/37 .031" leaded solder recommended)
* Phillips head screwdriver
* Flush side cutters (diagonal cutters)

Recommended (not included):
* No-clean flux paste (HIGHLY recommended to prevent bridging on the USB pins)
* Solder wick (to remove solder bridges if they occur)
* Solder sucker (to remove solder from holes if a component needs to be reinserted)
* Aluminium feet to angle the keyboard if desired
