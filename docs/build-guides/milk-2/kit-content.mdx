# Kit Content

Here's a list of parts needed for the build:

* 1 x [Milk 2% Kit](https://keebd.com/products/milk-2-keyboard-kit)

  Kit Includes:
  * 1 x 3D Printed Case
  * 1 x 2% Milk PCB
  * 4 x Mill-Max Sockets

Not included in the kit but required to complete:
* 1 x [Controller](https://keebd.com/collections/controllers)
* 1 x [USB Cable](https://keebd.com/collections/cables)
* 2 x [MX Switches](https://keebd.com/collections/mx-switches)
* 2 x [MX Keycaps](https://keebd.com/collections/mx-keycaps)
