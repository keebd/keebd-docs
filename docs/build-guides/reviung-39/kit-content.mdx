# Kit Content

The Reviung 39 is a compact 39-key (37.5%) column-staggered keyboard driven by a
Pro Micro. This is the **Mk-II** revision, which uses hotswap sockets (no
soldering your switches in permanently) and the larger, more heat-tolerant
WS2812B RGB LEDs for underglow and a single accent LED.

:::info Which version is this?
This guide covers the **Reviung 39 Mk-II**, the version we stock. It differs
from the original Reviung 39 (which used the smaller SK6812MINI LEDs) and from
the Reviung 39S (a "simple" through-hole variant with hand-soldered switches).
The Mk-II builds almost identically to the [Reviung 41](../reviung-41/).
:::

Here's a list of parts needed for the build:

* 1 x [Reviung 39 Mk-II Keyboard Kit](https://keebd.com/products/reviung-39-mk-ii-keyboard-kit)

  Kit Includes:
  * 1 x Reviung 39 Main PCB
  * 1 x Top Plate (PCB, with switch cut-outs)
  * 1 x Silicon Sheet
  * 1 x Upper Middle Plate (acrylic)
  * 1 x Lower Middle Plate (acrylic)
  * 1 x Bottom Plate (acrylic)
  * 1 x Pro Micro Protection Plate (acrylic)
  * 3 x Acrylic Spacers
  * 11 x Metal Spacers (7mm)
  * 22 x M2 Screws (4mm)
  * 3 x M2 Screws (10mm)
  * 3 x M2 Nuts
  * 6 x Rubber Feet
  * 39 x 1N4148 SMD Diodes
  * 39 x PCB Hotswap Sockets
  * 1 x Tactile Reset Switch
  * 11 x WS2812B RGB LEDs
  * 1 x Stabiliser (2u)

Not included in the kit but required to complete:
* 1 x [Pro Micro Controller](https://keebd.com/collections/controllers)
* 2 x Conthrough Spring-Pin Headers (12-pin) — used to socket the Pro Micro
* 39 x [MX-style Switches](https://keebd.com/collections/switches)
* [Keycaps for MX Switches](https://keebd.com/collections/caps): 36 x 1U, 1 x 2.25U, 2 x 1.25U
* 1 x Micro-USB Cable

Tools required:
* Soldering iron and solder (Kester 63/37 .031" leaded solder recommended)
* Fine tweezers (for placing SMD diodes, LEDs and sockets)
* Phillips head screwdriver
* Flush side cutters (diagonal cutters)

Recommended (not included):
* No-clean flux paste (highly recommended for the SMD LEDs and diodes)
* Solder wick (to remove solder bridges if they occur)
* Isopropyl alcohol and a lint-free cloth (to clean flux residue)
