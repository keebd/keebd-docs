# Kit Content

The Reviung 41 is a compact 41-key column-staggered keyboard driven by a Pro
Micro. It's a surface-mount, hotswap build with WS2812B RGB LEDs and an acrylic
diffuser that lights up the space beneath the board. The build is nearly
identical to the [Reviung 39 Mk-II](../reviung-39/), just with two extra keys.

Here's a list of parts needed for the build:

* 1 x [Reviung 41 Keyboard Kit](https://keebd.com/products/reviung-41-keyboard-kit)

  Kit Includes:
  * 1 x Reviung 41 Main PCB
  * 1 x Top Plate (PCB, with switch cut-outs)
  * 1 x Bottom Plate (acrylic)
  * 1 x Acrylic Diffuser
  * 1 x Pro Micro Protection Plate (acrylic)
  * 3 x Acrylic Spacers
  * 11 x Metal Spacers (7mm)
  * 22 x M2 Screws (4mm)
  * 3 x M2 Screws (10mm)
  * 3 x M2 Nuts
  * 6 x Rubber Feet
  * 41 x 1N4148 SMD Diodes
  * 41 x PCB Hotswap Sockets
  * 1 x Tactile Reset Switch
  * 11 x WS2812B RGB LEDs
  * 1 x Stabiliser (2u)

Not included in the kit but required to complete:
* 1 x [Pro Micro Controller](https://keebd.com/collections/controllers)
* 2 x Conthrough Spring-Pin Headers (12-pin) — used to socket the Pro Micro
* 41 x [MX-style Switches](https://keebd.com/collections/switches)
* [Keycaps for MX Switches](https://keebd.com/collections/caps): 36 x 1U, 1 x 2.25U, 4 x 1.25U
* 1 x Micro-USB Cable

Optional (not included):
* Middle acrylic plates and a silicon sheet, if you'd like the same layered
  case as the Reviung 39

Tools required:
* Soldering iron and solder (Kester 63/37 .031" leaded solder recommended)
* Fine tweezers (for placing SMD diodes, LEDs and sockets)
* Phillips head screwdriver
* Flush side cutters (diagonal cutters)

Recommended (not included):
* No-clean flux paste (highly recommended for the SMD LEDs and diodes)
* Solder wick (to remove solder bridges if they occur)
* Isopropyl alcohol and a lint-free cloth (to clean flux residue)
