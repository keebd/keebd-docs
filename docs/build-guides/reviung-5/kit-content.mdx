# Kit Content

The Reviung 5 is a tiny 4–5 key macropad driven by a Pro Micro controller. It
uses hand-soldered switches with leaded (through-hole) diodes, and has optional
support for per-key RGB underglow and a rotary encoder — so you can build it as
plainly or as fully-featured as you like.

Here's a list of parts needed for the build:

* 1 x [Reviung 5 Keyboard Kit](https://keebd.com/products/reviung-5-keyboard-kit)

  Kit Includes:
  * 1 x Reviung 5 PCB
  * 1 x Top Plate (acrylic)
  * 1 x Bottom Plate (acrylic)
  * 1 x Pro Micro Protection Plate (acrylic)
  * 3 x Acrylic Spacers
  * 3 x Metal Spacers
  * 3 x M2 Screws (long, for the top/bottom plates)
  * 6 x M2 Screws (short, for the protection plate)
  * 3 x M2 Nuts
  * 4 x Rubber Feet
  * 5 x 1N4148 Diodes
  * 1 x Tactile Reset Switch

Not included in the kit but required to complete:
* 1 x [Pro Micro Controller](https://keebd.com/collections/controllers)
* 2 x Conthrough Spring-Pin Headers (12-pin) — used to socket the Pro Micro
* 4–5 x [MX-style Switches](https://keebd.com/collections/switches)
* 4–5 x [1U Keycaps for MX Switches](https://keebd.com/collections/caps)
* 1 x Micro-USB Cable

Optional (not included):
* Up to 4 x WS2812B RGB LEDs (per-key lighting)
* 1 x Rotary Encoder (Alps EC12-compatible) + 1 x 1N4148W SMD diode

Tools required:
* Soldering iron and solder (Kester 63/37 .031" leaded solder recommended)
* Phillips head screwdriver
* Flush side cutters (diagonal cutters)
* Tweezers (needed if fitting the optional SMD LEDs)

Recommended (not included):
* No-clean flux paste (highly recommended if you fit the optional LEDs)
* Solder wick (to remove solder bridges if they occur)
* Isopropyl alcohol and a lint-free cloth (to clean flux residue)
