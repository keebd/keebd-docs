# ⚙️ Config Software

## VIAL
Vial is a feature-rich open-source cross-platform (Windows, Linux and Mac) GUI and a QMK fork for configuring your keyboard in real time.

- [VIAL Web Config](https://vial.rocks/)
- [VIAL Desktop Download](https://get.vial.today/download)

## VIA
VIA Configurator is an app that talks to your VIA-enabled keyboard and allows you to remap keys on-the-fly, as well as toggling layout options, programming macros, and controlling lighting.

- [VIA Web Config](https://usevia.app/)
- [VIA Desktop Download](https://github.com/the-via/releases/releases)
